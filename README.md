# Multi Location Weather Widget for Garmin Smart Watches

Available in the [Garmin connect IQ store](https://apps.garmin.com/apps/dcc2672a-b7e9-4c5e-a693-785fd4cff63d).

Show the weather conditions at your favourite locations as a widget on your Garmin smart watch.
The data is sourced from [the YR weather service](https://www.yr.no/en).
